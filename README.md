# slog

一个简单的头文件，用于捕获函数执行的异常并记录日志。

## 使用

```cpp
#define _DEBUG
#include "slog.h"

/* 普通函数 */
// int b = func(vec, a);
int b = SL(func, vec, a);

/* 成员函数 */
// Class cls
// cls.func(a, b, c);
SLC6(cls, Class::func, a, b, c);

/* 在函数中 */
float func(float a, float b)
{
  SSF
    return a / b;
  SEF(0.0)  // 失败返回的默认值
}
```
